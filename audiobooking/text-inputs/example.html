Example Article

By Example Author

This is an example article file that demonstrates the format used for scraped articles.

The actual article content will be placed in this div with class "article-content".

All other files in this directory are ignored by Git, but this example file is tracked.

Article Structure

Articles typically contain:

  • A title as an H1 heading
  • The author in a span with class "author" (which can be easily hidden with CSS if needed)
  • The content in this div
  • Various HTML elements like paragraphs, headings, lists, etc.

To generate more article files, run:

npm run scrape-articles
